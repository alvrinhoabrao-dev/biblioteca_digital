
# Relatório de Feedback

## Objetivo

Registrar observações e melhorias realizadas durante o desenvolvimento do sistema.

## Melhorias Implementadas

### Organização dos documentos

Foi criada uma funcionalidade de listagem organizada por tipo de documento e ano de publicação.

### Controle de duplicidade

Foi adicionada validação para impedir a criação de documentos com nomes já existentes.

### Busca de documentos

Foi implementada uma funcionalidade para localizar documentos pelo nome.

### Centralização das funcionalidades

Foi desenvolvido um menu principal para facilitar o acesso às operações do sistema.

## Conclusão

As melhorias implementadas aumentaram a organização, usabilidade e segurança do sistema.
