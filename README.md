# Sistema de Gerenciamento de Biblioteca Digital

## Sobre o Projeto

Este projeto foi desenvolvido em Python com o objetivo de simular um sistema de gerenciamento de biblioteca digital por meio de uma interface de linha de comando.

O sistema permite cadastrar, consultar, organizar e remover documentos digitais, além de gerenciar diretórios utilizados para armazenamento.

O desenvolvimento utilizou conceitos fundamentais de Python, incluindo modularização, funções, estruturas condicionais, laços de repetição, manipulação de arquivos e gerenciamento de diretórios.

---

## Funcionalidades

### Gerenciamento de Documentos

- Listar documentos
- Listar documentos organizados por tipo e ano de publicação
- Buscar documentos
- Adicionar documentos
- Ler documentos de texto (.txt)
- Renomear documentos
- Remover documentos

### Gerenciamento de Diretórios

- Criar diretórios
- Listar diretórios
- Remover diretórios
