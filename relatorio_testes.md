
# Relatório de Testes

## Objetivo

Validar o funcionamento das funcionalidades implementadas no sistema de gerenciamento de biblioteca digital.

## Testes Realizados

- Adição de documentos
- Listagem de documentos
- Listagem organizada por tipo e ano
- Busca de documentos
- Leitura de documentos
- Renomeação de documentos
- Remoção de documentos
- Criação de diretórios
- Listagem de diretórios
- Remoção de diretórios

## Resultado

Todos os testes apresentaram comportamento esperado e funcionamento adequado dentro do escopo do projeto.

## Conclusão

O sistema encontra-se apto para utilização conforme os requisitos propostos na atividade.
